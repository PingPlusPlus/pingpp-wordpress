== 安装 ==

将 pingpp 目录放到 /wp-content/plugins/ 下, 登录 wordpress 后台启用插件即可

== 配置 ==

登录 https://dashboard.pingxx.com, 将对应的 keys 填入插件配置页面即可

== 使用 == 
```
    [pingpp amount=100]
    [pingpp amount=100 channel="alipay_pc_direct"]
    [pingpp amount=100 selectable=true]
```
